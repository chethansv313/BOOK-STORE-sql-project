# Book Store Management and Sales Analytics Project

## Overview
This project is built using PostgreSQL.

It manages:
- Books
- Customers
- Orders

## Database Tables
- BOOKS
- CUSTOMERS
- ORDERS

## Concepts Used
- Primary Keys
- Foreign Keys
- Joins
- Aggregate Functions
- GROUP BY
- HAVING
- Business Analytics Queries

## Sample Reports
- Total Revenue
- Most Frequently Ordered Book
- Genre-wise Sales
- Customer Spending Analysis
- Stock Remaining Analysis

## Tools
- PostgreSQL
- pgAdmin
