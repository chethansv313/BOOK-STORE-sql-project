# Book Store Management and Sales Analytics Project

## Overview

This project is built using PostgreSQL to manage books, customers, and orders in a bookstore database.

## Database Tables

### BOOKS

* Book ID
* Title
* Author
* Genre
* Published Year
* Price
* Stock

### CUSTOMERS

* Customer ID
* Name
* Email
* Phone
* City
* Country

### ORDERS

* Order ID
* Customer ID
* Book ID
* Order Date
* Quantity
* Total Amount

## SQL Concepts Used

* Primary Keys
* Foreign Keys
* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions
* INNER JOIN
* LEFT JOIN

## Business Insights Generated

* Books by genre
* Revenue analysis
* Most frequently ordered book
* Customer spending analysis
* Genre-wise sales analysis
* Remaining stock after orders

## Tools Used

* PostgreSQL
* pgAdmin
* CSV Data Import
